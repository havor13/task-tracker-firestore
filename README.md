# Task Tracker (Cloud Databases Module)

## 📌 Project Overview
This project is a **Task Tracker application** built with **React** and **Firebase Firestore**.  
It demonstrates full **CRUD functionality** (Create, Read, Update, Delete) plus an **advanced search feature**.  
The app allows users to:
- Add new tasks
- View all tasks
- Edit existing tasks
- Delete tasks
- Search tasks by title

---

## 🚀 Features
- Firebase Firestore integration
- Modular React components (TaskForm, TaskList, SearchBar)
- Full CRUD operations
- Search/filter functionality
- Clean, responsive UI with CSS styling

---

## 🛠 Tech Stack
- **Frontend:** React 18
- **Database:** Firebase Firestore
- **Styling:** CSS
- **Package Manager:** npm
